## Project Overview
SmartExam is a mobile application designed to **simplify and unify the examination experience** for students. It supports **exam registration, attendance tracking, and answer submission** through a user-friendly interface.

The system includes:
- **Backend:** Node.js + Express + MongoDB
- **Frontend:** React Native + Expo
- **Cloud Integration:** MongoDB Atlas for database storage
- **Features:** User authentication, exam management, attendance marking, answer submission.
